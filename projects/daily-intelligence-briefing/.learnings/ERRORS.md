# Daily Briefing System - Error Log

Track errors for debugging and improvement.

---

## Format

```markdown
## [ERR-YYYYMMDD-XXX] component_name

**Logged**: ISO-8601 timestamp
**Priority**: high
**Status**: pending

### Summary
Brief description of what failed

### Error
```
Actual error message
```

### Context
What was happening when it failed

### Suggested Fix
How to resolve
```
