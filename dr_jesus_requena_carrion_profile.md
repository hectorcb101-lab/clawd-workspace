## PhyAAt Dataset Project

### Overview
**PhyAAt (Physiology of Auditory Attention to Speech)** - A major open-source dataset project for studying auditory attention using physiological signals.

### Resources
- **GitHub Organization**: [https://github.com/PhyAAt](https://github.com/PhyAAt)
- **Project Website**: [https://phyaat.github.io](https://phyaat.github.io)
- **Modeling Examples**: [https://phyaat.github.io/modeling/](https://phyaat.github.io/modeling/)
- **Papers With Code**: [https://paperswithcode.com/dataset/phyaat](https://paperswithcode.com/dataset/phyaat)

### Dataset Details
- **Signals Collected**: EEG, GSR (Galvanic Skin Response), PPG (Photoplethysmography)
- **Experiment Type**: Auditory attention on natural speech
- **Ethical Approval**: Queen Mary Ethics of Research Committee
- **Open Source**: Yes, with Python scripts for preprocessing, visualization, artifact removal, and predictive modeling

### Related Repositories
1. [PhyAAt/Dataset](https://github.com/PhyAAt/Dataset) - Main database
2. [PhyAAt/PredictiveModeling](https://github.com/PhyAAt/PredictiveModeling) - ML models
3. [PhyAAt/Visualization](https://github.com/PhyAAt/Visualization) - EEG visualization tools
4. [PhyAAt/ArtifactRemovalAlgorithm](https://github.com/PhyAAt/ArtifactRemovalAlgorithm) - Wavelet-based artifact removal
5. [PhyAAt/TextResponseAnalysis](https://github.com/PhyAAt/TextResponseAnalysis) - Statistical analysis
